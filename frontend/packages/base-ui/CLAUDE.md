# packages/base-ui — shadcn/ui 基础组件

## 模块职责

基于 Radix UI + Tailwind CSS 的 shadcn/ui 组件库。提供可复用的无样式原子组件。

## 组件列表

| 文件 | 组件 |
|---|---|
| `badge.tsx` | Badge 徽标 |
| `button.tsx` | Button 按钮 (variant: default/destructive/outline/secondary/ghost/link) |
| `calendar.tsx` | Calendar 日历 (react-day-picker) |
| `dialog.tsx` | Dialog 弹窗 |
| `hover-popover.tsx` | HoverPopover 悬停弹窗 |
| `input.tsx` | Input 输入框 |
| `label.tsx` | Label 标签 |
| `popover.tsx` | Popover 弹出框 |
| `select.tsx` | Select 下拉选择 |
| `skeleton.tsx` | Skeleton 骨架屏 |
| `switch.tsx` | Switch 开关 |
| `tabs.tsx` | Tabs 选项卡 |
| `toast.tsx` | Toast 消息提示 |

## 工具

- `cn.ts` — `cn()` 工具函数 (clsx + tailwind-merge)
