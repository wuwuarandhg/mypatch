# packages/biz-ui — 业务组件

## 模块职责

业务领域组件, 包含 K 线图、深度分析弹窗、AI 建议标签等。

## 组件列表

| 文件 | 组件 |
|---|---|
| `AmbientBackground.tsx` | 环境背景动画 |
| `InteractiveKline.tsx` | 交互式 K 线图 |
| `KlineModal.tsx` | K 线图弹窗 |
| `kline-indicators.tsx` | K 线技术指标 |
| `kline-summary-dialog.tsx` | K 线摘要弹窗 |
| `deep-analysis-modal.tsx` | 深度分析结果弹窗 |
| `stock-insight-modal.tsx` | 个股分析弹窗 |
| `logs-modal.tsx` | 系统日志弹窗 |
| `onboarding.tsx` | 新手引导 |
| `ai-suggestion-badge.tsx` | AI 建议标签 |
| `badge-chip.tsx` | 标签 Chip 组件 |
| `suggestion-badge.tsx` | 建议徽标 |
| `suggestion-action.ts` | 建议操作工具 |
| `technical-badge.tsx` | 技术分析徽标 |
| `price-alert-form-dialog.tsx` | 价格提醒表单弹窗 |
| `stock-price-alert-panel.tsx` | 价格提醒面板 |
